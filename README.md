# Linux Automation Scripts

Intro

## Scripts

| Script | Description |
|:-------|:------------|
| `create_user.sh` | Adds a new user, creates a home directory, sets permissions |


## How to Run

```bash
cd scripts
bash scriptname.sh
```

Make sure scripts are executable:
```bash
chmod +x scriptname.sh
```

## Requirements

- Linux system
- Bash shell
- sudo privileges (for user management and SSH setup)
